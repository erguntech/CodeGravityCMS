## Genel Bakış
CodeGravity CMS; slider, haber, ürün, proje, marka, referans, medya galerisi, firma bilgileri ve ziyaretçi takibi gibi modülleri tek panel üzerinden yönetmek için hazırlanmıştır.
Bu proje, özellikle cPanel shared hosting ortamlarında kolay deploy edilebilecek şekilde yapılandırılmıştır.

## Özellikler
- Admin Panel Üzerinden İçerik Yönetimi
- Slider Yönetimi
- Haber Yönetimi
- Ürün ve Ürün Kategorileri
- Proje ve Proje Kategorileri
- Marka Yönetimi
- Referans Yönetimi
- Medya Galerisi
- Firma Bilgileri Yönetimi
- API Tabanlı İçerik Paylaşımı
- Ziyaretçi Takip Sistemi
- Laravel Cache ve Storage Desteği
- cPanel Deploy Uyumlu Yapı
